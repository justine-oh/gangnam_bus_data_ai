# 강남구 버스 혼잡도 분류 AI 실습 자료

서울시 공공데이터를 활용해 **데이터 탐색 → 레이블링 → 분류 모델 학습**까지의 흐름을 직접 경험해보는 교육용 실습 자료입니다.  
모델 성능보다 전체 파이프라인을 따라가며 AI 개발 과정을 이해하는 것에 목적이 있습니다.

---

## 공공데이터 출처

**서울 열린데이터광장** 제공 (공공누리 제1유형)

- 버스노선별 정류장별 시간대별 승하차 인원 정보 (2025년 12월, 2026년 01월)
- 버스정류소 위치정보

---

## 구성

```
practice/
├── data/                           # 공공데이터 원본 (CSV, XLSX)
├── eda_practice/
│   ├── bus_eda_template.ipynb      # 시간대별 패턴 시각화, 전처리, 혼잡도 레이블 생성
│   └── bus_merged.csv              # 병합·정제된 데이터
└── model_train_practice/
    ├── bus_model_template.ipynb    # Random Forest 학습, 예측 테스트, ONNX 변환
    └── bus_labeled.csv             # 혼잡도 레이블(여유·보통·혼잡)이 붙은 학습 데이터
```

---

## 흐름

1. **EDA** — 두 달치 승하차 데이터를 합쳐 시간대별 패턴을 파악하고, 혼잡도 기준을 직접 설정해 레이블을 붙입니다.
2. **모델 학습** — 시간대(`hour`)와 정류장 번호(`stop_id`)를 입력으로, 혼잡도 등급(0·1·2)을 예측하는 분류 모델을 학습합니다.
3. **ONNX 변환** — 학습된 모델을 웹 브라우저에서 바로 실행할 수 있는 ONNX 포맷으로 변환합니다.
